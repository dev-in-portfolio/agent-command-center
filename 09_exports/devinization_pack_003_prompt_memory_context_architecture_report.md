## Operating Doctrine
The Agent Command Center does not treat memory as one bucket. Conversation context, active project state, user preferences, command language, project archives, cross-project patterns, and master archive memory must remain separated but connected. The whole agency remains available for context and recall, but active prompts receive only the relevant memory slice needed for the current task.
